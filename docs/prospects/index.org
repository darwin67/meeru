* Prospects
This directory holds roadmap summaries, phase-level outcome docs, risk registers, and longer-range backlog items that help guide sequencing but are not execution plans.

Execution-focused work stays under [[file:../plans/][docs/plans]]. Prospects summarize where those plans are intended to lead and which broader themes remain open.

** MVP First
Meeru should reach a working desktop email MVP before broader provider coverage, automation, or AI work. The intended MVP slice is:

- one generic account
- bounded IMAP sync into local storage
- folder list, message list, and reader UI
- basic compose and SMTP send
